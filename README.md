# euler_phi
Three different algorithms solving euler function. 

Method 1 ) Defination

Method 2 ) Use formula below: 
            - phi(mn) = phi(m) * phi(n)
            - phi(p^k) = p ^ k - p ^ (k - 1)
            
Method 3 ) Use formula below: 
            - phi(n) = n * (1 - 1/p1) * ... * (1 - 1/pn)
